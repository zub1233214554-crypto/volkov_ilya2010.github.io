Быстрая прокачка Skyrim

Оружие и доспехи

Кузнечное дело

  • Начинай с крафта железных кинжалов и мечей, покупая железные слитки и кожу у торговцев.
  • Собирай двемерский металлолом в руинах для изготовления двемерских луков и брони.
  • Используй зелья и зачарования на усиление навыка для большего бонуса.
  • Спи в своей постели для +10% к опыту, используйте камень-хранитель Воина для +20%.
  • Фарм сундуков и создавай предметы на верстаке для быстрого прогресса.

Тяжёлая броня

  • Используй локации с великанами для безопасного фарма опыта в тяжелой броне.
  • Бонусы от сна и еды ускоряют прокачку навыка.
  • Не перемещайся слишком часто, чтобы бонусы не сбрасывались.
  • Сочетай с лечением и регенерацией выносливости для максимального апа.
  • Комбинируй с квестами, где много врагов, для быстрого опыта.

Лёгкая броня

  • Тренируйся на драуграх в курганах, используя бонусы к скорости и лечению.
  • Используй заклинания и зелья для регенерации здоровья.
  • Сон с супругом дает +15% к опыту.
  • Постоянно улучшай легкую броню на верстаке для максимального бонуса.
  • Сочетай с навыками скрытности для безопасного фарма опыта.

Блок

  • Используй щит для защиты от атак врагов в локациях с великанами и драуграми.
  • Низкая сложность ускоряет прокачку навыка.
  • Комбинируй с лечением и регенерацией выносливости.
  • Используй зелья и супы для восстановления сил в бою.
  • Тренируй блок в сочетании с атакой для максимального опыта.

Одноручное оружие

  • Используй мечи и кинжалы против слабых врагов для быстрого опыта.
  • Комбинируй с зачарованиями для увеличения урона.
  • Фарм врагов в безопасных местах.
  • Прокачивай навык на тренировочных манекенах и NPC.
  • Используй усиление от зелий и еды для ускорения роста.

Двуручное оружие

  • Применяй двуручное оружие против сильных врагов для быстрого апа.
  • Сочетай с силовыми заклинаниями или зачарованной броней.
  • Используй тренировку с союзниками для безопасного фарма.
  • Регулярно повторяй квесты с массовыми врагами.
  • Используй сон и еду для бонусов к опыту.

Дальний бой (луки)

  • Тренируйся на мишенях или врагах на расстоянии.
  • Комбинируй с зачарованными стрелами и луками.
  • Используй бонусы от зелий и еды для ускоренного опыта.
  • Сон и отдых повышают получаемый опыт.
  • Повторяй квесты с дальними боями для стабильного апа навыка.

Магия

Разрушение

  • Используй заклинания разрушения по врагам для опыта.
  • Комбинируй с зачарованными предметами и зельями.
  • Учись у магов и через книги для бонуса опыта.
  • Фарм массовых врагов ускоряет рост навыка.
  • Следи за магической регенерацией для непрерывного апа.

Восстановление

  • Применяй заклинания лечения и зелья на союзников и себя.
  • Сон в безопасном месте повышает опыт.
  • Используй зачарования для усиления эффектов восстановления.
  • Комбинируй с квестами, где необходимо лечить NPC.
  • Повторяй действия регулярно для стабильного апа навыка.

Иллюзия

  • Используй заклинания страха, спокойствия или слепоты на врагов.
  • Комбинируй с зачарованной экипировкой для усиления эффекта.
  • Фармируй слабых врагов для опыта.
  • Используй тренировки в колегии магии для ускоренного апа.
  • Повторяй квесты с массовыми врагами для опыта.

Призывание

  • Призывай существ для помощи в бою для опыта.
  • Комбинируй с зачарованными предметами.
  • Учись у магов и через книги для ускоренного апа.
  • Фармируй врагов с помощью призываемых существ.
  • Повторяй квесты для стабильно роста навыка.

Изменение

  • Используй заклинания усиления защиты и оружия.
  • Комбинируй с зачарованными предметами.
  • Учись у магов для опыта.
  • Фармируй врагов с усилением заклинаний.
  • Повторяй тренировки и квесты для роста навыка.

Зачарование

  • Зачаровывай оружие и броню для бонусов к навыку.
  • Используй книги и столы зачарования.
  • Комбинируй с крафтом и зелиями для ускоренного апа.
  • Фармируй материалы для зачарования.
  • Повторяй процесс для максимального опыта.

  • Сочетай с невидимостью и зачарованиями.

  • Используй лёгкую броню для ускорения апа.
  • Повторяй квесты и тренировки для стабильного роста навыка.

Карманные кражи

  • Используй бонусы от еды и сна для ускоренного опыта.
  • Тренируйся на NPC в безопасных местах.
  • Комбинируй с зачарованием и экипировкой для успеха.
  • Повторяй действия и квесты для стабильного апа.

  • Комбинируй с зачарованными предметами для бонуса.

  • Практикуйся на разных типах замков для стабильного прогресса.

Воровство

  • Используй скрытность для успешных действий.
  • Комбинируй с зачарованной экипировкой для бонуса.
  • Повторяй задания для стабильно роста навыка.
  • Используй сон и бонусы еды для ускорения прокачки.

Ловушки

  • Активируй ловушки для врагов для опыта.
  • Учись правильно ставить и избегать их для успеха.
  • Комбинируй с зачарованием для усиления эффекта.
