Оружие и броня

Одноручное оружие

  • Используй мечи и кинжалы против слабых врагов для быстрого опыта.
  • Комбинируй с зачарованиями для увеличения урона и опыта.
  • Тренируйся на манекенах и NPC для безопасного фарма.
  • Повторяй квесты с массовыми врагами для стабильного роста навыка.
  • Используй зелья и еду для ускоренного повышения.

Двуручное оружие

  • Применяй двуручное оружие против сильных врагов для быстрого апа.
  • Сочетай с силовыми заклинаниями или зачарованной броней.
  • Используй тренировку с союзниками для безопасного фарма.
  • Повторяй квесты с массовыми врагами для стабильного опыта.
  • Используй сон и еду для бонусов к опыту.

Луки

  • Тренируйся на мишенях или врагах на расстоянии.
  • Комбинируй с зачарованными стрелами и луками.
  • Используй бонусы от зелий и еды для ускоренного опыта.
  • Сон и отдых повышают получаемый опыт.
  • Повторяй квесты с дальним боем для стабильного апа.

Блок

  • Используй щит для защиты от атак врагов в локациях с великанами и драуграми.
  • Низкая сложность ускоряет прокачку навыка.
  • Комбинируй с лечением и регенерацией выносливости.
  • Используй зелья и супы для восстановления сил.
  • Тренируй блок вместе с атакой для максимального опыта.

Лёгкая броня

  • Тренируйся на драуграх и слабых врагах для безопасного апа.
  • Используй бонусы от зелий, еды и сна.
  • Комбинируй с навыками скрытности для ускоренного опыта.
  • Постоянно улучшай броню на верстаке для бонуса.
  • Повторяй квесты и фарм локации для стабильного роста навыка.

Тяжёлая броня

  • Используй локации с великанами для опыта.
  • Сон и еда ускоряют прокачку навыка.
  • Не перемещайся слишком часто, чтобы бонусы не сбрасывались.
  • Комбинируй с лечением и регенерацией выносливости.
  • Сочетай с квестами для быстрого опыта.

Кузнечное дело

  • Начинай с крафта железных кинжалов и мечей.
  • Собирай руду и металлолом для улучшения брони и оружия.
  • Используй зелья и зачарования для бонусов.
  • Сон в собственной постели даёт +10% опыта, камень-хранитель Воина +20%.
  • Повторяй крафт и фарм материалов для максимального апа.

Магия

Разрушение

  • Используй заклинания разрушения по врагам для опыта.
  • Комбинируй с зачарованными предметами и зельями.
  • Учись у магов и через книги для бонуса опыта.
  • Фарм массовых врагов ускоряет рост навыка.
  • Следи за магической регенерацией для непрерывного апа.

Восстановление

  • Применяй заклинания лечения и зелья на союзников и себя.
  • Сон в безопасном месте повышает опыт.
  • Используй зачарования для усиления эффектов восстановления.
  • Комбинируй с квестами, где необходимо лечить NPC.
  • Повторяй действия регулярно для стабильного апа навыка.

Иллюзия

  • Используй заклинания страха, спокойствия или слепоты на врагов.
  • Комбинируй с зачарованной экипировкой для усиления эффекта.
  • Фармируй слабых врагов для опыта.
  • Используй тренировки в колегии магии для ускоренного апа.
  • Повторяй квесты с массовыми врагами для опыта.

Призывание

  • Призывай существ для помощи в бою для опыта.
  • Комбинируй с зачарованными предметами.
  • Учись у магов и через книги для ускоренного апа.
  • Фармируй врагов с помощью призываемых существ.
  • Повторяй квесты для стабильно роста навыка.

Изменение

  • Используй заклинания усиления защиты и оружия.
  • Комбинируй с зачарованными предметами.
  • Учись у магов для опыта.
  • Фармируй врагов с усилением заклинаний.
  • Повторяй тренировки и квесты для роста навыка.

Зачарование

  • Зачаровывай оружие и броню для бонусов к навыку.
  • Используй книги и столы зачарования.
  • Комбинируй с крафтом и зелиями для ускоренного апа.
  • Фармируй материалы для зачарования.
  • Повторяй процесс для максимального опыта.
